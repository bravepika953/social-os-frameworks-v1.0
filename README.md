# 社会OS分析フレームワーク v1.0  
**― 正しさではなく「構造」で動く社会の解剖図 ―**

社会は「正しいかどうか」ではなく、  
**どのような構造で処理されているか**によって結果が決まります。

このリポジトリは、社会現象を OS（基盤ロジック）として分解するための  
**6レイヤー構造の分析フレームワーク**を、視覚的に体験できる形でまとめたものです。

GitHub Pages 版では、各レイヤーを開きながら  
「社会がどのように変形されていくのか」を直感的に理解できます。

## 🧭 フレームワーク概要（6 Layers）

| Layer | 名称 | 役割 |
|-------|------|------|
| 6 | 出力（Visible Reality） | 社会に表出する現象。炎上・分断・政策結果など「見える世界」。 |
| 5 | 均衡（System Equilibrium） | 各プレイヤーの最適行動が作る“変わらない理由”。 |
| 4 | 制度フィルター（Institution Filter） | 理念が現実の制約（票・金・組織維持）で変形される層。 |
| 3 | 動員OS（Mobilization OS） | 社会を動かす形式。共感・敵味方・理念・物語の動員構造。 |
| 2 | 翻訳エンジン（Narrative Engine） | 生の現実を「物語」へ圧縮する層。意味づけ・解釈・編集。 |
| 1 | 入力（Raw Input） | 怒り・希望・不安など、社会の“生の感情”。 |

## 🎯 このフレームワークの目的
- 社会現象を「構造」で理解する  
- 個々の事件ではなく、**OS（基盤ロジック）**を捉える  
- 責任の所在が曖昧になりがちな現代社会で、  
  **どこで歪みが生まれ、どこで止まるのか**を可視化する  
- 思想・政治・制度を“階層”として扱うことで、  
  感情論ではなく構造論で議論できるようにする
